Курс: Создание приложений на Node.js
Тема: Введение. Модули

Каждое задание рассматривайте как отдельные проекты.

Задание 1
=========
Модифицируйте код для ввода из stdin, созданный на занятии, чтобы получилась удобная функция prompt().
а) Функция должна быть реализована в отдельном модуле ES6.
б) Функция должна принимать в качестве аргумента текст приглашения (как prompt в браузере).
в) Функция должна возвращать экземпляр класса Promise.

//Пример использования № 1:
prompt("Сколько вам лет?")
.then(Number)
.then(val => userAge = val)

//Пример использования № 2:
let userIn  = await prompt("Сколько вам лет?")
let userAge = Number(userIn)


Задание 2
=========
Модифицируйте сделанную функцию prompt, чтобы она принимала второй необязательный аргумент - значение по умолчанию, которое функция должна вернуть при некорректном вводе, и которое обозначает тип данных, в который нужно преобразовать то что введёт пользователь.
Примеры:

let userAge = await prompt("Сколько вам лет?", 5)
// так как второй аргумент - число типа Integer, функция должна преобразовать данные пользователя в целое число. Если преобразование не удастся или пользователь ничего не ввёл, функция должна вернуть 5.

let userWeight = await prompt("Какой у вас вес?", 0.0)
// второй аргумент - Float. Функция должна вернуть данные пользователя в формате Float, либо значение 0.0, если преобразование прошло с ошибкой или пользователь ничего не ввёл.

let userName = await prompt("Как вас зовут?", "no name")
// второй аргумент - String. Функция должна вернуть строку, которую ввёл пользователь, либо "no name", если он не ввёл ничего.

Кроме указанных вариантов второй аргумент может быть типа Boolean или Array.
